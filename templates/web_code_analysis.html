معلومات حول النظام

يقوم النظام باكتشاف الثغرات الأمنية في كود الويب باستخدام تقنيات الذكاء الاصطناعي والتحليل الثابت.

نصائح للاستخدام

  • تأكد من صحة الكود المدخل قبل التحليل
  • لتحليل موقع ويب، أدخل رابطاً مباشراً للصفحة
  • يمكن رفع ملفات HTML/PHP بحد أقصى 16MB
  • النتائج قد تستغرق بضع ثوانٍ حسب حجم الكود

أنواع الثغرات المكتشفة

  • حقن SQL (SQL Injection)
  • ثغرات XSS
  • انكشاف معلومات حساسة
  • مشكلات إدارة الجلسات
  • ثغرات CSRF
